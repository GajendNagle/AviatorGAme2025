How to play

1 Minutes 1 issue, 45 seconds to order, 15 seconds waiting for the draw. It opens all day. The total number of trade is 1440 issues.

If you spend 100 to trade, after deducting 5 service fee, your contract amount is 95.

1. Select Green: If the result shows 1,3,7,9 you will get (95*2) 190. If the result shows 5, you will get (95*1.5) 142.5

2. Select Red: If the result shows 2,4,6,8 you will get (95*2) 190. If the result shows 0, you will get (95*1.5) 142.5

3. Select Violet: If the result shows 0 or 5, you will get (95*4.5) 427.5

4. Select Number: If the result is the same as the number you selected, you will get (95*9) 855

5. Select Big: If the result shows 5,6,7,8,9 you will get (95 * 2) 190

6. Select Small: If the result shows 0,1,2,3,4 you will get (95 * 2) 190
